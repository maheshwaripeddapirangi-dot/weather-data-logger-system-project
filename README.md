# 🌦️ Weather Data Logger System

A Python + MySQL project that fetches real-time weather data using WeatherAPI and stores it in a database.

---

## 🚀 Features
- Get live weather data by city
- Store data in MySQL database
- View weather history
- View last search
- Logging system for debugging

---

## 🛠️ Technologies Used
- Python
- MySQL
- WeatherAPI
- Requests library
- Logging module

---
